# Configurações do Neovim

Este repositório contém minhas configurações personalizadas para o Neovim. Utilizo uma abordagem modular para gerenciar plugins, empregando o Packer para facilitar a instalação e atualização.

## Plugins Principais

### 1. Packer.nvim

Packer.nvim é um gerenciador de plugins rápido, eficiente e flexível para o Neovim. Facilita a gestão de plugins, permitindo a especificação e instalação fácil de extensões.

### 2. Lspconfig e cmp

lspconfig é utilizado para configurar servidores de linguagem (LSP - Language Server Protocol) para fornecer funcionalidades avançadas, como linting, sugestões e navegação inteligente. cmp é um plugin de auto-completar altamente configurável, compatível com LSP.

### 3. Telescope

Telescope é um poderoso explorador de arquivos para o Neovim, facilitando a navegação e busca por arquivos, buffers, entre outros.

### 4. Solarized-Osaka

Solarized-Osaka é o tema principal utilizado nestas configurações, proporcionando uma experiência visual agradável e de alto contraste.

## Requisitos

Certifique-se de que os seguintes requisitos estejam instalados em seu sistema antes de utilizar estas configurações:

Node.js: Necessário para alguns plugins que dependem de componentes JavaScript.
Compilador C: Algumas extensões e plugins podem requerer a compilação de código C.
